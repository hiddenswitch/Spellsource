General Information

This information is used on all card types.

"id": The internal identifier for the card. This must be unique, so that cards
	can be distinguished from another card.
"name": The name of the card. This is the displayed name on the card, and does
	not have to be unique.
"baseManaCost": This is the base mana cost for the card before any mana
	modification.
"type": The type of card the card is. There are several different types of
	cards:
		MINION
		SPELL
		CHOOSE_ONE
		WEAPON
		HERO_POWER
		HERO
"heroClass": The class that the card belongs to or ANY if the card is neutral.
"rarity": What rarity the card is. These rarities are:
		FREE
		COMMON
		RARE
		EPIC
		LEGENDARY
"description": The card text that is displayed on the card.
"manaCostModifier": The mana cost modifier that changes how much the card costs
	to play.
"collectible": Whether the card is collectible or not.
"set": Which card set the card belongs to. Currently, these are:
		CLASSIC
		NAXXRAMAS
		GOBLINS_VS_GNOMES
		BLACKROCK_MOUNTAIN
		THE_GRAND_TOURNAMENT
		LEAGUE_OF_EXPLORERS



Minion Information

This information is used only on minion cards.

"baseAttack": The base amount of attack the minion has.
"baseHp": The base amount of health the minion has.
"race": The race of the minion. Currently, these are:
		BEAST
		DEMON
		DRAGON
		MECH
		MURLOC
		NONE
		PIRATE
		TOTEM
"battlecry": The spell that is cast when this minion is played.
"deathrattle": The spell that is cast when this minion is destroyed.
"aura": The aura spell that is granted by this minion.
"trigger": The spell that is casted when the appropriate trigger occurs. See
	triggers.txt for a list of triggers.
"attributes": The list of attributes that a minion has. See attributes.txt for a
	list of attributes.
